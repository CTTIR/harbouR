# Environment variables read by harbouR

Environment variables read by harbouR

## Environment variables

- `SEATABLE_SERVER` — default server URL used by
  [`hb_client()`](https://r-heller.github.io/harbouR/reference/hb_client.md).

- `SEATABLE_API_TOKEN` — default API token used by
  [`hb_client()`](https://r-heller.github.io/harbouR/reference/hb_client.md).

These are read at call time; you can change them between calls without
restarting R.
